A hidden input field let web developers include data that cannot be seen or modified by users when a form is submited.A hidden field often stores what database record that needs to be updated when the form is submitted.
NOTE: While the value is not displayed to the user in the page's content ,it is visible(and can be edited) using any browser's developer tools or "View Source " functionality.Do not use hidden input as a form of security!
